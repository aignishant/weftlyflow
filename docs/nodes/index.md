# Built-in nodes

*Populated as nodes land. Phase-1 tier-1 nodes ship first.*

See `IMPLEMENTATION_BIBLE.md §25` for the full tiering.

## Tier 1 — MVP core (~24 nodes)

Control flow, data manipulation, execution control, triggers. First to ship.

## Tier 2 — popular integrations (~40)

Slack, Discord, GitHub, Google Workspace, Notion, Airtable, Stripe, HubSpot, ...

## Tier 3 — long tail (~240)

Everything else. Added on demand, one PR per integration.

## AI (~15 to start)

OpenAI, Anthropic, Google, Ollama, Mistral; ReAct / function-calling agents; buffer / summary / window memory; pgvector / qdrant vector stores; chat trigger.
